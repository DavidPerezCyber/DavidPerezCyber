# CrackStation

## ¿Qué es?

CrackStation es un servicio online que permite identificar contraseñas a partir de hashes utilizando grandes bases de datos precalculadas.
Utilizarlo a través de https://crackstation.net/.

---

## ¿Cuándo utilizarlo?

Cuando durante una auditoría se obtiene un hash y se desea comprobar si corresponde a una contraseña conocida.

---

## Metodología

1. Obtener el hash.
2. Identificar el algoritmo.
3. Buscar el hash en CrackStation.
4. Validar el resultado obtenido.

---

## Errores comunes

- Intentar utilizarlo con hashes poco comunes o protegidos mediante algoritmos robustos.
- No identificar correctamente el tipo de hash.

---

## Buenas prácticas

Utilizar primero CrackStation antes de invertir tiempo en ataques de fuerza bruta o diccionario.
