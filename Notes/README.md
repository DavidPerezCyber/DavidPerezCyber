# 📚 Notes

Bienvenido a mi repositorio de apuntes.

Este espacio reúne todo el conocimiento que voy adquiriendo durante mi formación en ciberseguridad ofensiva y mientras preparo la certificación OSCP.

No pretende ser una copia de documentación oficial ni una recopilación de teoría de Internet. El objetivo es documentar mi propio aprendizaje utilizando mis propias palabras, organizando la información de forma clara para poder consultarla rápidamente siempre que la necesite.

Aquí encontrarás explicaciones técnicas, metodologías, comandos útiles, ejemplos prácticos y herramientas utilizadas durante laboratorios, máquinas vulnerables y proyectos personales.
