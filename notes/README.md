# 📚 Notes

Bienvenido a mi repositorio de apuntes.

Este espacio reúne todo el conocimiento que voy adquiriendo durante mi formación en ciberseguridad ofensiva y mientras preparo la certificación OSCP.

No pretende ser una copia de documentación oficial ni una recopilación de teoría de Internet. El objetivo es documentar mi propio aprendizaje, utilizando mis propias palabras, de forma que cualquier concepto pueda revisarlo rápidamente cuando lo necesite.

Aquí encontrarás apuntes prácticos, metodologías, comandos útiles, ejemplos reales y pequeñas explicaciones que me ayuden a comprender mejor cada técnica.
