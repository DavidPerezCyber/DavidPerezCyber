# 🏢 Active Directory

Este repositorio reúne toda la documentación relacionada con Active Directory y los distintos ataques, técnicas de enumeración y metodologías utilizadas durante auditorías de entornos Windows.

El objetivo es construir una base de conocimiento organizada que me permita comprender tanto el funcionamiento interno de Active Directory como las técnicas ofensivas empleadas por un pentester durante una auditoría.

Toda la documentación se basa en laboratorios prácticos, máquinas vulnerables y preparación para la certificación OSCP.

## Contenido

- Enumeración
- Kerberos
- LDAP
- SMB
- BloodHound
- Privilege Escalation
- Lateral Movement
- Persistence
- Mitigaciones
- Herramientas

Cada técnica incluirá explicaciones, comandos, ejemplos prácticos y referencias para facilitar su consulta.
